# 茶言觀色 · 客戶流失風險評估器

一個完整的前端網站，用於評估茶飲店客戶流失風險，具有亮色/深色主題切換功能。

## 功能特色

### 🎨 雙主題系統
- **亮色主題**：清新茶綠色調，適合日間使用
- **深色主題**：專業深色調，減少眼睛疲勞
- **一鍵切換**：右上角主題切換按鈕
- **自動偵測**：根據系統偏好自動選擇主題
- **本地儲存**：記住用戶的主題偏好

### 📊 風險評估功能
- **客戶資料輸入**：年齡、性別、區域、會員月數等基本資料
- **消費行為分析**：到訪頻率、消費金額、滿意度評分等
- **即時風險計算**：基於輸入數據計算流失風險分數
- **視覺化儀表板**：圓形進度條顯示風險等級
- **風險因子分析**：列出關鍵風險因素及建議

### 🎯 互動功能
- **表單驗證**：即時輸入驗證與錯誤提示
- **重置功能**：一鍵重置表單至預設值
- **動畫效果**：平滑過渡與互動反饋
- **響應式設計**：適配各種螢幕尺寸
- **通知系統**：操作成功/錯誤提示

## 技術架構

### 前端技術

## 使用方式

### 快速開始
1. 直接開啟 `index.html` 檔案於瀏覽器中
2. 或使用本地伺服器運行：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js
   npx serve .
   ```

### 主題切換
- 點擊右上角的月亮/太陽圖標按鈕
- 系統會記住您的選擇，下次訪問時自動應用

### 風險評估流程
1. 在左側表單輸入客戶資料
2. 點擊「評估風險」按鈕
3. 查看右側的風險分析結果
4. 根據建議採取相應行動

## 開發指南

### 自訂主題
修改 `index.html` 中的 Tailwind 配置：
```javascript
tailwind.config = {
    darkMode: "class",
    theme: {
        extend: {
            colors: {
                // 自訂色彩...
            }
        }
    }
}
```

### 擴充功能
1. **後端整合**：連接API進行真實數據分析
2. **數據匯出**：匯出風險報告為PDF/CSV
3. **多語言支援**：添加英文/日文等語言
4. **進階圖表**：使用Chart.js或D3.js可視化
